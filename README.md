## ポートフォリオ：SDGsに特化したSNSアプリケーション「MySDGs」

活動の輪を広げ、地球をより豊かにすることが目的。  
「MySDGs」は、個人やコミュニティ単位で行っているSDGs活動の発信、  
イベントやチャット機能による交流を通して  
SDGsの普及や活動促進を目的としたアプリケーション。
